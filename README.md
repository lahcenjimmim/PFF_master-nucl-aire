# Étude des isotopes du plomb avec le formalisme RHB

## Description du projet

Ce projet s'inscrit dans le cadre d'une étude en physique nucléaire théorique, utilisant le formalisme **Relativistic Hartree-Bogoliubov (RHB)** pour analyser les propriétés des isotopes du plomb (Pb).

L'objectif principal est d'étudier l'évolution de différentes grandeurs physiques en fonction du nombre de neutrons, notamment :

* l'énergie de liaison
* l'énergie d'appariement
* l'énergie de désintégration alpha
* l'énergie de séparation à deux neutrons (S2n)
* les rayons de charge
* les effets de fermeture de couches

## Méthodologie

Les calculs sont réalisés à l'aide du code **DIRHB**, basé sur des interactions relativistes telles que :

* DD-ME2
* DD-PC1

Les résultats sont ensuite traités et analysés à l’aide de **Jupyter Notebook**, permettant :

* la visualisation graphique des données
* la comparaison entre différents modèles
* l'interprétation physique des résultats

## Grandeurs physiques étudiées

* **Énergie de liaison** : mesure de la stabilité globale du noyau
* **Énergie d’appariement** : liée aux corrélations entre nucléons, particulièrement importante pour décrire les noyaux ouverts
* **Énergie de désintégration alpha** : permet d'étudier la stabilité vis-à-vis de l’émission α
* **Énergie de séparation à deux neutrons (S2n)** : indicateur des transitions de structure nucléaire
* **Rayon de charge** : information sur la distribution spatiale des protons
* **Fermetures de couches** : signatures des nombres magiques et de la structure en couches du noyau

## Objectifs scientifiques

Ce travail vise à :

* comprendre la structure nucléaire des isotopes du plomb
* identifier les signatures de fermeture de couches
* analyser la stabilité nucléaire via la désintégration alpha
* étudier les effets d'appariement dans la structure nucléaire
* comparer différents modèles d'interaction nucléaire
